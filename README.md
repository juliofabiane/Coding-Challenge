# Coding-Challenge
Coding Challenge for Coderocrk

## Dependências
	Node js
	npm

## Baixar o Repositório
	SSH: 
	git clone git@github.com:juliofabiane/CodingChallenge.git
	
	Ou
	HTTPS: 
	git clone https://github.com/juliofabiane/CodingChallenge.git
	
	cd CodingChallenge

## Instalação
	npm install -g gulp
	npm install	
	gulp
	
	http://localhost:3000
